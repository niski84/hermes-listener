🔔 Wake word (optional sidecar)

Leave blank to transcribe everything. Configure to gate transcription on a wake word.

▸ Install openWakeWord sidecar 
# Quick start (requires Python 3.10+, pip)
pip install --user openwakeword
# Or use the dedicated sidecar from 2nd-whisper-brain:
git clone https://github.com/niski84/2nd-whisper-brain.git ~/wakeword-src
cd ~/wakeword-src/wake_word_sidecar
./start.sh   # default port 9201

👤 Speaker filter (optional sidecar)

ECAPA voiceprint matching. Drops clips that don't sound like you — filters family, TV, podcasts.

▸ Install ECAPA speaker sidecar 
# SpeechBrain + FastAPI sidecar from 2nd-whisper-brain
cd ~/wakeword-src/speaker_sidecar
pip install --user -r requirements.txt
./start.sh   # default port 9200
# Then enroll your voice:
curl -F "audio=@your-30-second-sample.wav" http://localhost:9200/enroll
